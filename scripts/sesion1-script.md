¡Hola! Bienvenidos a esta primera sesión del curso Fundamentos de Java. Hoy vamos a descubrir qué es programar, qué es un lenguaje de programación y por qué Java es tan importante. Y al final, haremos juntos nuestros primeros programas. ¡Empecemos!


Programar es darle instrucciones a una computadora para que haga lo que queremos.
Es como explicarle paso a paso cómo resolver un problema o realizar una tarea. Cada vez que usamos una app, un videojuego o una página web, hay un programador detrás que escribió el código para que todo funcione. Podríamos decir que programar es transformar ideas en acciones que una computadora puede entender.


En programación hay tres palabras que vamos a escuchar mucho. Primero, el código, que son las instrucciones que entiende la computadora. Luego, el programador, que es quien analiza el problema y escribe la solución. Y finalmente, la sintaxis, que son las reglas del lenguaje, igual que la gramática en un idioma. Si rompemos esas reglas, el programa no funcionará. Así que aprender la sintaxis es como aprender el idioma de la máquina.


Un lenguaje de programación es la forma en que los humanos nos comunicamos con las computadoras. Cada lenguaje tiene sus propias palabras y reglas, pero todos sirven para lo mismo: dar instrucciones claras. Por ejemplo, aquí vemos el mismo mensaje escrito en dos lenguajes distintos. En Python se escribe de una manera más corta, y en Java un poco más detallada. Aun así, los dos hacen exactamente lo mismo: mostrar un texto en pantalla. Así que, aunque cambie la forma de escribir, la idea detrás es siempre la misma: decirle a la computadora qué hacer.


Java es un lenguaje de programación creado en 1995 por Sun Microsystems. Desde entonces, se ha vuelto uno de los más populares del mundo porque es seguro, estable y funciona en casi cualquier sistema. Con Java se pueden hacer muchas cosas: páginas web, aplicaciones móviles, videojuegos, programas de empresas y mucho más. Por eso, aprender Java es abrir la puerta a muchísimas oportunidades en el mundo de la tecnología.


Ahora que ya sabemos qué es Java, hablemos de por qué es tan importante y sigue siendo uno de los lenguajes más usados del mundo. Java destaca porque combina tres cosas esenciales. Primero, es seguro y confiable. Millones de empresas lo usan para manejar información importante sin riesgo de errores graves o ataques. Segundo, es multiplataforma, lo que significa que un mismo programa puede funcionar en Windows, macOS, Linux o incluso en celulares Android, sin necesidad de reescribir el código. Y tercero, tiene una gran comunidad de desarrolladores en todo el mundo. Si tienes un problema, probablemente alguien ya lo resolvió y compartió la respuesta. Por eso, Java está presente en casi todo: desde aplicaciones bancarias y sistemas universitarios, hasta videojuegos, cajeros automáticos y herramientas de inteligencia artificial. Así que cuando aprendes Java, no solo aprendes a programar: estás aprendiendo un lenguaje que te servirá en muchísimos campos de la tecnología.


Para este curso vamos a usar JDoodle como nuestro compilador de Java. JDoodle es una página web que nos permite escribir y ejecutar código sin instalar nada en la computadora, solo con el navegador.


La primera forma de entrar es muy sencilla. En el navegador abrimos Google y escribimos jdoodle. En los resultados, buscamos la opción que dice ‘Online Java Compiler’ y hacemos clic allí. Esa página nos lleva directo a un editor de Java listo para usar: solo tenemos que escribir nuestro código y presionar Run para verlo funcionar.


La segunda forma es entrar desde la página principal de JDoodle. Igual que antes, en Google escribimos jdoodle y esta vez hacemos clic en el primer resultado que dice ‘JDoodle – Free AI powered Online Coding Platform’. Al abrir la página, vamos a la barra de búsqueda de lenguajes y escribimos java. En la lista que aparece, elegimos Java. Se abrirá una ventana para crear un nuevo proyecto. Aquí dejamos seleccionado Single-file, elegimos la versión de Java que viene por defecto y marcamos Empty Project, para comenzar con un archivo vacío. Luego hacemos clic en Start Coding, y listo: ya tenemos nuestro espacio para escribir programas en Java.


# Ejercicio 1

Primero escribimos `public class Main {`. Esto crea una clase llamada Main. En Java, todo empieza dentro de una clase. Luego escribimos el método principal:
`public static void main(String[] args) {` Este método es el punto de inicio del programa, aquí es donde Java empieza a ejecutar el código. Ahora, dentro de main, agregamos esta línea: `System.out.println("¡Hola, Java!");` Con esto le pedimos a la computadora que muestre ese texto en la consola. Agregamos una segunda línea para otro mensaje, y al final cerramos las llaves de `main` y de la clase `Main`.

# Ejercicio 2

Vamos a hacer ahora un segundo ejercicio: una pequeña tarjeta con información.
Para eso, vamos a modificar el código que ya tenemos. De nuevo empezamos con la clase `Main y el método main`. Primero imprimimos una línea de título para nuestra tarjeta. Aquí hacemos lo mismo con el curso. Cerramos la tarjeta con una línea de igualdades y luego cerramos las llaves del método y de la clase. 


# Mini Reto

Antes de terminar la sesión, te dejo un mini reto para que practiques lo que vimos hoy.

En el PASO 1, vas a crear una clase y el método principal.
Algo como lo que ves aquí: `public class MyClass {}` y dentro `public static void main(String[] args) {}`.
Esa es la estructura básica de cualquier programa en Java.

En el PASO 2, escribe un título para tu presentación usando `System.out.println("=== Mi Presentación ===");`.
Puede ser ese texto u otro que tú quieras.

Y en el PASO 3, agrega tres datos sobre ti usando también `System.out.println();`.
Por ejemplo, tu nombre, tu hobby y una meta que tengas.

La idea es que al ejecutar el programa, en la consola se vea una pequeña tarjeta de presentación hecha por ti.

Cuando termines, ya habrás creado tu primera mini presentación en Java.
