INTRODUCCIÓN
Bienvenidos a esta sesión del curso. Hoy vamos a aprender un tema fundamental para la programación: los métodos y la entrada y salida de datos en Java.
Los métodos nos permiten organizar mejor nuestro programa, evitar repetir instrucciones y mantener el código más claro.
La entrada y salida de datos, por otro lado, nos permite interactuar con el usuario, pedir información y mostrar resultados.

QUE ES UN MÉTODO
Un método es un bloque de código que realiza una tarea específica. Cada método tiene un nombre y contiene instrucciones que se ejecutan cuando ese método es llamado.
Usar métodos nos ayuda a dividir el programa en partes más pequeñas y fáciles de entender. También permite reutilizar código en diferentes partes del programa sin volver a escribirlo.

PARÁMETROS Y VALOR DE RETORNO
Los métodos pueden recibir datos. Esos datos se llaman parámetros.
Además, algunos métodos pueden devolver un resultado. A esto se le llama valor de retorno.
Los parámetros permiten que el método trabaje con información variada. El valor de retorno permite que un método genere un resultado que puede usarse en otra parte del programa.

TIPOS DE MÉTODOS
Existen principalmente tres tipos de métodos:

Métodos sin retorno:
Realizan una acción, pero no devuelven ningún valor.

Métodos con retorno:
Devuelven un resultado que puede ser utilizado más adelante en el programa.

Métodos estáticos:
Se pueden llamar sin crear un objeto. El método main es un ejemplo de un método estático.

ENTRADA Y SALIDA DE DATOS
Para recibir información del usuario utilizamos la clase Scanner.
La salida de datos se realiza usando instrucciones que muestran texto en pantalla.
Combinando entrada y salida con métodos podemos crear programas interactivos que respondan al usuario.

EJERCICIO PRÁCTICO
A continuación, vamos a explicar cómo realizar un ejercicio práctico que reúne todos los conceptos vistos.
El ejercicio consiste en lo siguiente:

Pedir el nombre del usuario.

Llamar a un método encargado de mostrar un saludo personalizado.

Pedir dos números al usuario.

Llamar a un método encargado de sumar esos dos números y devolver el resultado.

Mostrar el resultado final en la pantalla.

EXPLICACIÓN DEL EJERCICIO
Primero se prepara un método independiente que reciba el nombre del usuario y muestre un saludo.
Luego se prepara un segundo método que reciba dos valores numéricos y devuelva la suma.
En el método principal se utiliza Scanner para pedir el nombre y los dos números.
Después se llama al método que saluda al usuario usando el nombre que ingresó.
Luego se llama al método que suma los dos números y se guarda el resultado.
Finalmente, se muestra el resultado al usuario.

CIERRE
Con esto el estudiante aprende a:

Crear métodos

Enviar información a un método mediante parámetros

Recibir información desde un método mediante un retorno

Usar Scanner para interactuar con el usuario

Organizar mejor el código separando responsabilidades

Esta sesión sienta las bases para programar de manera más clara y modular. Es importante practicar creando más métodos y combinándolos con entrada y salida de datos para afianzar lo aprendido.
